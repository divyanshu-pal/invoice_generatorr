## For Frontend

Go to client

**initialize**

command:- npm i

To run frontend code

command:- npm run dev

## For Backend(set your mongodb url in the .env file)
Go to backend 

**initialize**
command:- npm i

To run backend code

command:- npm run build

After above command

npm start
